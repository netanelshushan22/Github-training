## 📌 על הפרויקט

הפרויקט הזה הוא בסיס לדשבורד אינטראקטיבי באמצעות הספרייה `Dash` של `Plotly`.
הקוד הראשוני נמצא בקובץ `dashboard.py` וניתן להרחיב אותו בהתאם לצרכים שלך (גרפים, פילטרים, טבלאות ועוד).

---

## 🛠 טכנולוגיות וכלים

- **שפת תכנות**: Python  
- **פריימוורק Frontend**: Dash (`dash`, `dash.html`, `dash.dcc`)  
- **ניהול קוד**: Git + GitHub  

---

## 🚀 התחלה מהירה (Quick Start)

### 1. שכפול (Clone) של המאגר

```bash
git clone <URL-של-המאגר-בגיטהאב>
cd "Github training"
```

### 2. יצירת סביבת עבודה (מומלץ)

```bash
python -m venv .venv
source .venv/bin/activate  # ב-Windows: .venv\Scripts\activate
```

### 3. התקנת חבילות

```bash
pip install dash
```

*(אם יש לך קובץ `requirements.txt` – עדכן את הפקודה בהתאם)*  

### 4. הרצת הדשבורד

```bash
python dashboard.py
```

לאחר ההרצה, פתח דפדפן וגלוש לכתובת שתופיע בטרמינל (לרוב `http://127.0.0.1:8050`).
